# Design System — MoviUpVote

## Product Context
- **What this is:** Уютный сайт-топ фильмов и сериалов для тусовки друзей. Вход через Google, голосование ▲, лучшее всплывает наверх.
- **Who it's for:** Небольшой доверенный круг своих (5-30 человек), не открытый интернет.
- **Space/industry:** Кино/медиа, соц-список рекомендаций (соседи по духу: Letterboxd, но домашний и тёплый).
- **Project type:** Web app — одна читаемая колонка, ранжированная лента.
- **Memorable thing:** «Уютное место своих — тёплое, со вкусом к кино, не корпоративное.» Каждое решение служит этому.

## Aesthetic Direction
- **Direction:** Тёплый редакторский минимализм — «печатный киножурнал для своих» (Anthropic-grade: тихо, воздушно, бумага).
- **Decoration level:** minimal → intentional (тёплый бумажный фон, волосяные разделители; ноль теней, хрома, украшений).
- **Mood:** Тихо, тепло, со вкусом. Ощущение хорошей бумаги и литературного серива, а не «эппа».
- **Reference sites:** anthropic.com (тёплый крем + глиняный акцент), эстетика Tiempos/Copernicus (работали от знаний, без live-ресёрча).

## Typography
- **Display/Hero:** **Fraunces** (opsz, 500-700) — мягкий оптический литературный серив. Заголовки, masthead, названия тайтлов. Ближайший бесплатный родич Copernicus/Tiempos.
- **Body:** **Newsreader** (opsz, 400-600) — текстовый серив для чтения. Мета, кнопки, интерфейсный текст.
- **UI/Labels:** Newsreader (та же семья).
- **Data/Tables:** Newsreader с `font-variant-numeric: tabular-nums` — счётчики голосов не прыгают при пересортировке. ОБЯЗАТЕЛЬНО на числах.
- **Code:** (не применимо для UI; при необходимости — Geist Mono.)
- **Loading:** Google Fonts. `Fraunces:opsz,wght@9..144,400;9..144,500;9..144,600;9..144,700` + `Newsreader:opsz,wght@6..72,400;6..72,500;6..72,600`. С `display=swap`, `preconnect`.
- **Scale (px):** meta 14.5 · body 18 · UI 15-16 · title 22 · rank 20 · h1 44. Модульная ~1.2-1.25.

## Color
- **Approach:** restrained — тёплая пастель + один глиняный акцент. Цвет редкий и значимый.
- **Primary (accent):** `#C36B4E` (Clay/терракота) — голос ▲ активный, кнопка «Добавить», фокус. Единственный «горячий» цвет.
- **Secondary:** `#F0E1D6` (Voted, пастельный персик) — фон поднятого голоса; мягкое активное состояние.
- **Neutrals (тёплые):** Paper `#F4F1EA` (фон) · Surface `#FBF9F4` (строка/карточка) · Line `#E5DFD3` (волосяные разделители) · Muted `#837B6F` (таупе, мета) · Ink `#232019` (тёплый почти-чёрный, текст).
- **Semantic:** success `#5C7A5A` · warning `#C79A3E` · error `#B4573F` (в тон глине) · info `#5F7486`. Держать приглушёнными, в тёплой гамме.
- **Dark mode (тёплый):** Paper `#1C1915` · Surface `#242019` · Ink `#EDE7DA` · Muted `#A69C8C` · Clay `#D98A6A` · Voted `#3A2C22` · Line `#332D24`. Насыщенность акцента снижена ~15%. Через CSS custom properties + `data-theme`.

## Spacing
- **Base unit:** 8px.
- **Density:** spacious (воздух — фирменная черта эстетики).
- **Scale:** 2xs(2) xs(4) sm(8) md(16) lg(24) xl(32) 2xl(48) 3xl(64). Строки списка: ~18px вертикали. Колонка отделена ~40px от края.

## Layout
- **Approach:** grid-disciplined — одна читаемая центрированная колонка.
- **Grid:** одна колонка на всех брейкпоинтах; ряд = [rank] [vote] [info].
- **Max content width:** 680px.
- **Border radius:** sm 10px (инпуты, кнопки) · md 12px (кнопка голоса) · full 999px (пилюли/тумблеры). Без «пузырящегося» одинакового радиуса на всём.
- **Dividers:** волосяная линия `--line` между строками; никаких тяжёлых карточек и теней.

## Motion
- **Approach:** minimal-functional. Тихие фейды, мягкий переход состояния голоса. Никаких подпрыгиваний.
- **Easing:** enter `ease-out` · exit `ease-in` · move `ease-in-out`.
- **Duration:** micro 50-100ms · short 150-250ms (переход голоса ~180ms) · medium 250-400ms (смена темы ~300ms).

## Anti-patterns (не делать)
Фиолетовые градиенты, 3-колоночная сетка фич с иконками в кружках, центрирование всего подряд, sans-шрифт по умолчанию (мы серив-first), тяжёлые тени, gradient-кнопки, system-ui как основной шрифт.
